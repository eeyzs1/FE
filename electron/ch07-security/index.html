📋 安全七大原则

1上下文隔离 (Context Isolation) — Preload 与渲染进程运行在不同上下文，防止恶意代码访问特权 API
2禁用 Node.js 集成 — 渲染进程不应直接使用 require()，通过 contextBridge 安全暴露
3启用沙箱 — 限制渲染进程对系统资源的访问
4内容安全策略 (CSP) — 限制资源加载来源，防止 XSS
5不加载远程内容 — 优先加载本地文件，不禁用同源策略
6安全的 IPC — 不暴露整个 ipcRenderer，为每个通道创建封装
7验证输入 — 永远不信任渲染进程数据，防止注入攻击
