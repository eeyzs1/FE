⚡ 性能优化清单

✅ 使用 show: false + ready-to-show 避免白屏闪烁
✅ 设置 backgroundColor 减少视觉闪烁
✅ 设置 backgroundThrottling: false 禁用后台节流（适用于音乐播放器等需要在后台运行的场景）
✅ 使用 webview 隔离重型内容
✅ 延迟加载非关键窗口
✅ 使用 asar 打包减少文件 I/O
✅ 避免在渲染进程中执行 CPU 密集型任务
✅ 使用 UtilityProcess 处理 CPU 密集型任务
✅ 监听 visibilitychange 事件暂停不可见页面的操作
✅ 使用 requestAnimationFrame 替代 setInterval 进行动画

🧪 测试方法

单元测试 (Jest): 测试独立的函数和模块
E2E 测试 (Playwright): 测试完整的用户流程
主进程测试: 使用 electron-mocha 或 rewire 模拟
渲染进程测试: 使用 Jest + jsdom 或 Playwright
