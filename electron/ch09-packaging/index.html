🛠️ Electron Forge 打包流程

步骤 1：安装 Forge
pnpm add -D @electron-forge/cli npx electron-forge import
步骤 2：配置 forge.config.js
module.exports = { packagerConfig: { name: 'MyApp', icon: './assets/icon', // 不带扩展名 asar: true, }, makers: [ { name: '@electron-forge/maker-squirrel', config: { name: 'my_app' } }, { name: '@electron-forge/maker-zip', platforms: ['darwin'] }, { name: '@electron-forge/maker-deb', config: {} }, { name: '@electron-forge/maker-rpm', config: {} }, ], publishers: [ { name: '@electron-forge/publisher-github', config: { repository: { owner: 'your-username', name: 'your-repo' }, prerelease: false, }, }, ], }
步骤 3：打包
npx electron-forge package
步骤 4：生成安装包
npx electron-forge make

🔄 自动更新代码示例

在主进程中使用 autoUpdater 实现自动更新：

const { autoUpdater } = require('electron') // 配置更新源 autoUpdater.setFeedURL({ provider: 'github', owner: 'your-username', repo: 'your-repo', }) // 检查更新 autoUpdater.checkForUpdates() // 监听事件 autoUpdater.on('update-available', (info) => { console.log('发现新版本:', info.version) }) autoUpdater.on('download-progress', (progress) => { console.log(`下载进度: ${progress.percent.toFixed(1)}%`) }) autoUpdater.on('update-downloaded', () => { // 提示用户重启安装 autoUpdater.quitAndInstall() })

🔐 代码签名

Windows: 需要代码签名证书（EV 或 OV），可从 DigiCert、Sectigo 等购买。

macOS: 需要 Apple Developer 账号，使用 codesign 和 notarytool 签名和公证。

// macOS 签名和公证 codesign --deep --force --sign "Developer ID Application: Your Name" MyApp.app xcrun notarytool submit MyApp.zip --apple-id your@email.com --team-id TEAMID --password APP_PASSWORD
