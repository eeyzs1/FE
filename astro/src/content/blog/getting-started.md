# Getting Started with Astro

Welcome to your first blog post! In this article, we'll explore the fundamentals of Astro.

## What is Astro?

Astro is a modern web framework that delivers lightning-fast performance by shipping zero JavaScript by default.

## Key Features

- **Islands Architecture**: Interactive components load independently
- **Content Collections**: Type-safe content management
- **Framework Agnostic**: Use React, Vue, Svelte, or any framework

## Your First Component

```astro
---
const name = "World";
---
<h1>Hello, {name}!</h1>
```
